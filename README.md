# Portafolio Personal

## Características

- **Diseño moderno y oscuro** con excelente experiencia de usuario
- **Responsivo**
- **Navegación suave** con scroll
- **Interactividad avanzada** con JavaScript vanilla
- **Consumo de API** real (JSONPlaceholder)
- **Modal dinámico** de proyectos
- **Formulario de contacto** funcional
- **Animaciones** al hacer scroll
- **Menú móvil** con animaciones


## Tecnologías Utilizadas

- **HTML5**
- **Tailwind CSS** (vía CDN)
- **JavaScript** 
- **Font Awesome** para iconos
- **Google Fonts**
